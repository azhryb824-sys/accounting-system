# قاعدة التشغيل الجاهزة

هذا المشروع يستخدم SQLite، وهي مجانية ومناسبة للاستخدام التجاري حسب ترخيصها العام المفتوح.

تمت إضافة ملف `starter.sqlite3` كقاعدة بيانات نظيفة للتشغيل الأولي، بدون جلسات أو بيانات مستخدمين حقيقية.

## التشغيل على Windows

شغّل الملف:

```text
setup.bat
```

سيستخدم قاعدة `starter.sqlite3` ويشغّل الخادم المحلي.

بيانات الدخول الافتراضية:

```text
رقم الهوية: 2572280689
كلمة المرور: Admin@12345
```

غيّر كلمة المرور بعد أول دخول.

## تشغيل يدوي

```powershell
set SQLITE_NAME=starter.sqlite3
python manage.py runserver
```

## ملاحظات

- لا يتم رفع قاعدة `db.sqlite3` لأنها قاعدة عمل محلية.
- يمكن نسخ `starter.sqlite3` إلى `db.sqlite3` إذا أردت أن تصبح هي القاعدة الافتراضية.
- مفتاح Gemini لا يوضع داخل القاعدة أو الكود. استخدم `GOOGLE_API_KEY` كمتغير بيئة.
