# Arabic Voice Assets

This folder is reserved for licensed Arabic voice preview files.

Important:
- Browser speech still reads dynamic AI replies using `speechSynthesis`.
- Static `.wav` or `.mp3` files can only play fixed samples.
- For real dynamic Arabic voices, connect a server-side TTS engine or API.

Open model candidate:
- SILMA TTS: https://huggingface.co/silma-ai/silma-tts
- License shown for the model repository: Apache-2.0.

When adding files:
1. Place the audio file in this folder.
2. Update `voices.json`.
3. Use only files with a license that allows your intended use.
